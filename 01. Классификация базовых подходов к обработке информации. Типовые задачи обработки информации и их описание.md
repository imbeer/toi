
# Базовые подходы к обработке информации

## Статистический подход

основан на использовании: плотностей и функций распределения вероятностей, моментов распределений, достаточных статистик.

- Алгоритм реализует статистическое решающее правило.
- Качество обработки определяется с использованием статистических характеристик (вероятностей ошибок, дисперсий или матриц ковариации ошибок)

## Детерминистский подход

отказ от использования вероятностных моделей данных.

Виды:

| назание                    | описание                                                                                                                                                                         |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Метрический                | Использование метрик (функций расстояния);<br>Использование гипотез о свойствах объектов (гипотеза компактности и однородности классов образов).                                 |
| Нейросетевой               | Представление алгоритма в виде преобразователя "вход-выход" на основе ансамбля нейронов;<br>Требует обучения, подходит для сложных задач, где модель объекта не задана априорно. |
| Структурно-лингвистический | Использует подобразы как признаки и анализирует их отношения через "грамматику" образов;<br>Позволяет применять теорию формальных языков.                                        |

# Задача классификации (распознавания)

Отнесение объекта, описываемого образом (признаками), к одному из классов.
Количество классов и их эталонные описания известны или могут быть определены в процессе обучения.

1. Во многих случаях эталонное описание классов может быть получено без обучения на основе постулирования математической модели образов. Например, гауссовские шумы в радиосигналах.
2. В других случаях предполагается обучение на множестве прецедентов обучающей выборки объектов распознавания.

> [!note] распознавание с обучением на основе имеющегося множества прецедентов называют классификацией с обучением (с учителем).

# Задача кластеризации

Разбиение множества объектов (образов) на кластеры по степени сходства (близкие образы в одном кластере, отличающиеся — в разных).

1. число классов может быть известно;
2. число классов может быть неизвестно (автоматическая классификация, классификация без обучения).

Задача кластеризации при *не заданном количестве классов* является наиболее сложной для решения: высокое качество разбиения достигается только при большой степени различия образов разных классов.

# Задача оценивания

Определение неизвестных характеристик объекта на основе обработки совокупности первичных данных, представленных либо непосредственно как наблюдения, либо в виде признаков (отобранных наблюдений).

В качестве оцениваемых характеристик могут выступать самые различные величины, например:
- один из параметров состояния объекта
- вектор параметров, характеризующих полное состояние объекта $s\in{R^n}$

Виды:
- параметрическое оценивание (известная форма распределения);
- непараметрическое оценивание.

Выделяют **задачу фильтрации**: 
	оценивание непосредственно ненаблюдаемой реализации векторного случайного процесса (случайной функции времени) или реализации случайного поля (случайной функции двух и более переменных), описывающих изменения состояния объекта во времени или пространстве.

# Задача регрессии

Установление функциональной зависимости между ожидаемыми значениями зависимых (выходных) переменных и значениями независимых (входных) переменных.

Решение базируется на использовании обучающей выборки данных, в которой задается 
1. совокупность значений вектора независимых переменных $X^{N} = \{x^{(1)}, ...,x^{(N)}\}$; 
2. совокупность значений зависимой переменной – «ответов» $Y^{N} = \{y^{(1)}, ...,y^{(N)}\}$, где каждый элемент определен на множестве вещественных чисел.

Может рассматриваться несколько зависимых переменных, которые объединяются в вектор.
Полученная функциональная зависимость используется для предсказания ожидаемого значения зависимой переменной для нового набора входных данных.

> [!note] Если значение зависимой переменной определяется по отношению к будущему моменту времени, которое присутствует в векторе независимых переменных, то задача регрессии называется **задачей прогнозирования.**

# Задача ассоциаций

Нахождение значимых связей (ассоциаций) между объектами или событиями.

**Ассоциации** - определенные правила, которые могут быть использованы для объяснения природы анализируемых процессов и предсказания появления новых событий.

Поиск ассоциативных правил является одним из популярных приложений Data Minning, при реализации которого ищутся часто встречающиеся наборы объектов среди большого множества таких наборов.

> [!note] По сути, данная задача является частным случаем задачи классификации.
