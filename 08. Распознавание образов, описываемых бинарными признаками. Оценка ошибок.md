
# Бинарные признаки

Признаки (компоненты вектора признаков), которые могут принимать **только два возможных значения**.

Вектор признаков $\vec{x} = (\vec{x}_1, \ldots, \vec{x}_n)^T$ состоит из независимых компонентов, которые принимаю значения из двух элементов $x_i \in V = \{v_1, \dots, v_m\}, \quad k = \overline{1, n}$. В простейшем случае множество значений состоит из двух элементов $x_k \in V = \{0, 1\}$. 
Для каждого класса заданы априорные вероятности гипотез $\omega_i: p(\omega_j)$ и функции правдоподобия $p(x/\omega_i), \quad i = \overline{1, M}$ 

$$p(x / \omega_i) = \prod_{k=1}^n p(x_k / \omega_i), \quad i = \overline{1, M},$$

Плотность распределения общего вида с использованием дельта-функций Дирака: 

$$p(x_k / \omega_i) = \sum_{t=1}^m p_k(v_t / \omega_i) \delta(x_k - v_t), \quad \sum_{t=1}^m p_k(v_t / \omega_i) = 1, \quad k = \overline{1, n}, \quad i = \overline{1, M},$$

где $p_k(v_t / \omega_i)=p_{k,t,i}=p_{k,t}(\omega_i)$ -  различные обозначения вероятности получения значения $v_t$ для признака $x_k$ при условии, что образ принадлежит классу $\omega_i$.

Другая форма записи это дельта-функции Кронекера:

$$p(x_k / \omega_i) = \sum_{t=1}^m p_k(v_t / \omega_i) \tilde{\delta}(x_k - v_t), \quad \tilde{\delta}(x_k - v_t) =
\begin{cases} 
1, & x_k = v_t, \\
0, & x_k \neq v_t.
\end{cases}$$

При рассмотрении задачи распознавания классов с независимыми бинарными признаками (принимают значения с вероятностью 0 и 1), описание каждого класса и каждого признака используется только две вероятности: 

$$p_{k_i} = p(x_k = 1 / \omega_i), \quad 1 - p_{k_i} = p(x_k = 0 / \omega_i), \quad k = \overline{1, n}.$$

Полное вероятностное описание каждого класса имеет форму: 

$$p(x / \omega_i) = \prod_{k=1}^n p_{k,i}^{x_k} (1 - p_{k,i})^{1-x_k}, \quad i = \overline{1, M}.$$

## Случай двух классов

Вероятностей значений бинарных признаков двух классов 

$$p_k = p(x_k = 1/\omega_1), \quad 1 - p_k = p(x_k = 0/\omega_1), \quad k = \overline{1, n},$$

$$q_k = p(x_k = 1/\omega_2), \quad 1 - q_k = p(x_k = 0/\omega_2), \quad k = \overline{1, n}.$$

Функции правдоподобия классов и ЛОП: 

$$p(x / \omega_1) = \prod_{k=1}^n p_k^{x_k} (1 - p_k)^{1-x_k}, \quad p(x / \omega_2) = \prod_{k=1}^n q_k^{x_k} (1 - q_k)^{1-x_k},$$

$$g'(x) = \sum_{k=1}^n \ln \frac{p(x_k / \omega_1)}{p(x_k / \omega_2)} = \sum_{k=1}^n \left( x_k \ln \frac{p_k}{q_k} + (1 - x_k) \ln \frac{1 - p_k}{1 - q_k} \right) \;\overset{\omega_1}{\underset{\omega_2}{\gtrless}}\; l'_0.$$

здесь ЛОП и разделяющая функция $g'(x) - l'_0$ образуют линейную границу между классов в виде 

$$g'(x) - l'_0 = \sum_{k=1}^n \alpha_k x_k + \alpha_0 = 0,$$

$$\alpha_k = \ln \frac{p_k (1 - q_k)}{q_k (1 - p_k)}, \quad \alpha_0 = \sum_{k=1}^n \ln \frac{1 - p_k}{1 - q_k} - l'_0.
$$

# Расчет ошибок

Способы:
- приближенные соотношения для законов распределения g"(x);
- прямое имитационное моделирование алгоритма.

Случай вероятности единиц и нулей одинаковы: $p_k = p \neq 0, \quad q_k = q \neq 0, \quad k = \overline{1, n}$ 
Тогда: $g'(x) = L_x \ln \frac{p}{q} + P_x \ln \frac{1-p}{1-q} \;\overset{\omega_1}{\underset{\omega_2}{\gtrless}}\; l'_0, \quad L_x = \sum_{k=1}^n x_x, \quad P_x = \sum_{k=1}^n (1-x_x) = n-L_x,$
где $L_x$ -число единиц (ответов “да”), $P_x$ - число нулей (ответов “нет”).

При p > q, что эквивалентно 

$$L_x = \sum_{k=1}^n x_k \geq L_0 = \text{round} \left[ \left( l'_0 - n \ln \frac{1-p}{1-q} \right) / \ln \frac{p(1-q)}{q(1-p)} \right]$$

Ошибки распознавания используя схему Бернулли:

$$\alpha = \sum_{t=0}^{L_0-1} C_n^t p^t (1-p)^{n-t}, \quad \beta = 1 - \sum_{t=0}^{L_0} C_n^t q^t (1-q)^{n-t}$$

Примеры:
- модель распознавания бинарных изображений в условиях воздействия помех (вероятность искажения помехой , “перинверсия”)
- обнаружение импульсной последовательности сигналов (радиолокатор)
